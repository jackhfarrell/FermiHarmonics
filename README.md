# FermiHarmonics

FermiHarmonics is a Julia code for solving a harmonic-expansion form of the linearized 2D Boltzmann equation for Fermi-liquid transport, including momentum-relaxing and momentum-conserving collisions, on unstructured meshes with Trixi.jl.

![FermiHarmonics regime demo](demo/streamlines.png)

Documentation: <https://docs.jackhfarrell.com>
