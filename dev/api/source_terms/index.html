Physical Source Terms

FermiHarmonics.physical_sources — Function
physical_sources(u, x, t, equations) -> SVector

Source terms from a BGK-type approximation to the collision integral. We do not damp $a_0$ (density). The momentum modes $a_1, b_1$ are damped at the momentum-relaxing rate $\gamma_mr$, while higher harmonics are damped at the full scattering rate $\gamma_mr + \gamma_mc$.
